# Sheet Music Library Manager 

A full-stack web application to help individuals or organizations organize and manage their sheet music collections. Built as a portfolio project to practice full-stack development, relational databases, and authenticationâ€”while solving a practical problem for music management.

---

## Features

- **PDF Upload & Metadata Management**  
  Upload sheet music PDFs and automatically populate the title from the file name. Users can edit additional details such as composer, arranger, orchestrator, whether it contains parts, whether it contains the score, and more.

- **Performance Logging**  
  Track performances and associate them with sheet music documents using a many-to-many relationship.

- **User Authentication**  
  Secure login via OAuth2, allowing multiple users to manage their own collections.

- **Full-Stack Implementation**  
  Built with a relational database and integrated front-end and back-end functionality.

---

## Tech Stack

- **Backend:** Java, Spring Boot, Hibernate  
- **Frontend:** Thymeleaf, Vanilla JavaScript  
- **Database:** MySQL  
- **Authentication:** OAuth2
